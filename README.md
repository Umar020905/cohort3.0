# HTML Day 1 🚀

This repository contains my Day 1 HTML practice project as I begin my web development journey.

## What I Learned

* HTML document structure
* Headings (`h1` to `h6`)
* Paragraphs and text formatting
* Lists (`ol`, `ul`)
* Links and navigation
* Images, videos, and audio
* Tables
* Forms and input fields
* Dropdown menus
* Semantic HTML basics

## Features

* Different heading levels
* Text styling examples
* Ordered and unordered lists
* Interactive dropdown menu
* External links
* Image, video, and audio embedding
* Basic table creation
* User input form with various input types

## Technologies Used

* HTML5

## Project Goal

The goal of this project is to practice and understand the fundamentals of HTML and build a strong foundation for learning CSS and JavaScript.
